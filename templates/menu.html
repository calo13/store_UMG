Descripción de la Tarea

Esta tarea consiste en desarrollar una aplicación web para la administración de clientes y productos. Utilizaremos tecnologías como Phyton, Flask, Javascript Bootstrap, CSSy archivos de texto para el almacenamiento de datos.
